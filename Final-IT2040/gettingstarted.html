What kind of Raspberry Pi is right for you?

When it comes to selecting which raspberry pi is right for you, there are a lot of different factors to consider before deciding. To start, one thing to keep in mind when selecting a raspberry pi for your next project is size constraints. If you project requires you to be in a small form factor, the Pi Zero(w) may be a good option for you! However, if size isnt an issue, it could be advised to look at one of the larger more powerful Pis such as the 4B or 3B+. Listed below is a very basic core comparison between the 3 different raspberry Pis i can reccomend in 2020. In the following tutorials, I will personally be using a raspberry pi 4B+ (4gb of ram). This little machine will do just about any project you may want, ranging from a basic web server to even a full fledged minecraft server!

Version Price Core Count Cpu Clock Speed
Pi Zero(w) 10-15$ 1 core 1 Ghz
Pi 3B+ 35$ 4 cores 1.4 Ghz
Pi 4B 35-45$ 4 cores 1.5 Ghz

How to setup a raspberry pi

When setting up our raspberry pi, we are going to use a very light OS called "DietPi". DietPi is an extremely light weight version of the OS offered by the Raspberyy Pi organization Themselves. DietPi is reccomended for beginner use due to its extensive collection of tools and applications for easy setup and operation.

Required equipment

  • Raspberry Pi
  • 8gb Micro SD Card (minimum)
  • Wireless connection
  • Micro SD Card Reader
  • A monitor that can connect to the Pi

How to install DietPi

  1. To start, download the right image of DietPi for your corresponding device. In this tutorial, select the Raspbery Pi option.
  2. After you download the file, unzip the file using 7zip, winrar, or unarchiver(mac). Make sure you store this somewhere you will remember.
  3. To actually write this image to the SD card, we need to install software to do this. The reccomended software is called Balena Etcher. Balena Etcher is used for writing OS images to SD cards and USB Drives. In our case, we will use it to write to our SD Card.
  4. After installing Etcher, launch the program and locate the DietPi image in the software. Next, select the correct SD card to write the image to, and then hit Flash! This can take some time to finish flashing the card.
  5. If you are using Wi-Fi, browse the data on the SD card and open the file titled "Dietpi.txt". In this file, modify the line:

    AUTO_SETUP_NET_WIFI_ENABLED=0

    change this to:

    AUTO_SETUP_NET_WIFI_ENABLED=1

  6. To configure the wifi, open the file titled "dietpi-wifi.txt". Find and replace MySSID and MyWiFiKey to your network credentials. Once you do this, save and exit. You can now remove your SD card and put it in to your raspberry pi!
  7. Turn on the pi and connect it to your monitor. On first boot, follow the steps on screen and configure the software packages to your liking. Change the SSH client to OpenSSH to ensure the best compatibility with windows SSH protocols.
  8. After setting it up, the Pi will restart and will prompt you to log in. After this, it will ask you to change the password to something more secure. Now that we have our pi setup, take note of the IP address of the Pi.After that, it is now time to connect to it!

Connecting to the raspberry pi for the first time

Now that you have your Raspberry pi setup, and the IP of the Pi recorded, we are now going to connect to it via SSH in powershell! Connecting to it via SSH has a few benefits over giving it a dedicated monitor. For starters, your PI can now be placed anywhere in the house with no issues. You also no longer need 2 sets of peripherals to interact with the device.

  1. To start, ensure the Rasperry Pi is on and running. You will only be able to connect to the device when it is on.
  2. Launch powershell on your computer. You can use other programs such as Putty, but powershell has been the most consistent and reliable for my testing.
  3. To connect to the Pi, type "ssh -l root ip-of-the-pi". After this, you should be prompted to type in your password. If done correctly, you should now be connected to your raspberry pi via SSH!

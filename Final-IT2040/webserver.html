How to setup a basic web server on a Raspberry Pi

One major use for a Raspberry pi is the creation of a web server to host web content! By self hosting, you can save money monthly to host your project site/portfolio. Another major perk of self hosting is that you have sole control over what access you have on your server. You can even run multiple websites/applications off one device! In this tutorial, we are going to show you how to create a basic web server using a raspberry pi, based on the steps from our Getting Started tutorial.

How to setup the webserver

  1. To start, Start by SSHing into your raspberry pi. This can be done by typing "ssh -l root (your ip address)". Once you hit enter, the raspberry pi will then prompt you for your password.
  2. Now that we are connected to the Pi, we are going to want to type "dietpi-software". This will launch us into the installation page for different software packages.
  3. Once you are in the software menu, select the "Software Optimised" tab and press enter. You are going to want to scroll down until you find the package called LAMP. Press spacebar on this and then press enter.
  4. After this, tab down to the Install button and hit enter. Give it some time while everything downloads. Once everything is downloaded, the raspberry pi will prompt you to restart, go ahead and press ok.
  5. Once the Pi restarts reconnect via SSH again. The webserver should now automatically be running in the background!
  6. To test and see if the webserver is working, navigate to "http://youriphere" and click enter. If the default Apache2 webpage pops up, you have successfully installed the webserver!
  7. To change the file the webserver shows, navigate to /var/www/html by typing "cd /var/www/html" into the pi. By replacing the Index.html file, the content on the webserver will change!

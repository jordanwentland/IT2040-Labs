How to setup a basic Minecraft server on a Raspberry Pi

Whats a game that universally everyone has heard of, can enjoy, and relates to? Thats right! Minecraft! Minecraft is a game originally developer by mojang where a player is placed in a world to do as they want freely. However, sometimes playing alone can be kinda boring. So, this tutorial will teach you how to setup a Minecraft server on a raspberry pi so that you can play with your friends!

How to setup the Minecraft Server

  1. To start, Start by SSHing into your raspberry pi. This can be done by typing "ssh -l root (your ip address)". Once you hit enter, the raspberry pi will then prompt you for your password.
  2. Now that we are connected to the Pi, we are going to want to type "dietpi-software". This will launch us into the installation page for different software packages.
  3. Once you are in the software menu, select the "Software Optimised" tab and press enter. You are going to want to scroll down until you find the package called MineOS. Press spacebar on this and then press enter to install it.
  4. After this, tab down to the Install button and hit enter. Give it some time while everything downloads. Once everything is downloaded, the raspberry pi will prompt you to restart, go ahead and press ok.
  5. Once the Pi restarts reconnect via SSH again. The minecraft server should now automatically be running in the background!
  6. To connect to the web interface for the server, open a web browser and navigate to "https://yourserverip:8443". The webserver runs on port 8443, so if you do not specify this it will not connect.
  7. To log in, use the same credentials as on the raspberry pi. In my scenario since im using the default credentials, it would user=root and pass=dietpi.
  8. After this, you should successfully be in your minecraft web interface, happy crafting!
